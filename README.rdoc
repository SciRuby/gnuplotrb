== Installation
While gem isn't on rubygems, it may be installed from source.
=== Dependencies
You will require {gnuplot 5.0}[http://www.gnuplot.info/download.html] to use that gem.
=== Gem installation
  git clone https://github.com/dilcom/pilot-gnuplot.git
  cd pilot-gnuplot
  bundle install
  rake install
