== Installation
While gem isn't on rubygems, it may be installed from source.
=== Dependencies
* Ruby 2.0+
* It is required to install {gnuplot 5.0}[http://www.gnuplot.info/download.html] to use that gem.
=== Gem installation
==== Install without bundler
  git clone https://github.com/dilcom/pilot-gnuplot.git
  cd pilot-gnuplot
  bundle install
  rake install
==== Install using bundler
* add
    gem 'pilot-gnuplot', :git => 'https://github.com/dilcom/pilot-gnuplot.git'
  to your Gemfile
* run
    bundle install
